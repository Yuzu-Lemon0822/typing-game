# Typing Game

GitHub Pagesで動作するシンプルなタイピングゲームです。

## 特徴
- 漢字、ふりがな、ローマ字表示
- 柔軟なローマ字入力対応（しゃ -> sha, sya どちらも可）
- ミス時のアニメーション演出

## ディレクトリ構造
- `index.html`: エントリーポイント
- `main.js`: メインスクリプト
- `main/`: ゲームロジック
    - `game.js`: ゲーム進行管理
    - `data.js`: 問題データ・ローマ字変換マップ
    - `input.js`: キーボード入力管理

## 実行方法
このプロジェクトはES Modulesを使用しているため、ローカルで確認する場合はローカルサーバーが必要です。
